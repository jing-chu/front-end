Data Types

Number

The number type represents both integer and floating point numbers.

String

A string in JavaScript must be surrounded by: double quotes, single quotes or backticks.

Boolean

The boolean type has only two values: true and false.

Array

An array is a special variable, which can hold more than one value at a time.
syntax: var array_name = [item1, item2, ...];

Accessing an array item by referring to the index number. Array indexes start with 0.

Type Casting

to convert a value from a type to another

In general converting to a string: calling the toString() method on any value, or pass any value to the String() global function.

Casting to number by using the Number() global function, or unary + operator

Any value can be converted to boolean passing it to Boolean()

Operators

Arithmetic Operators

Arithmetic operators are used to perform arithmetic on numbers.

Assignment Operators

Assignment operators assign values to JavaScript variables.

Comparison Operators

The comparison operators take simple values (numbers or string) as arguments and evaluate either true or false.

Logical Operator

There are three logical operators in JavaScript: || (OR), && (AND), ! (NOT).
